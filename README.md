# A Simple Maze Game
A maze game for terminal/console. The program is written with C++.

## Project Requirements
* Showcasing any algorithm from below in the project

    - bfs
    - dfs
    - asymptotic
    - binary_search
    - counting inversion
    - dynamic programming
    - greedy
    - kmp
    - number theory
    - recurrence
    - scc
    - string matching rabin karp
    - merge sort
    - quick sort
    - topological sort
